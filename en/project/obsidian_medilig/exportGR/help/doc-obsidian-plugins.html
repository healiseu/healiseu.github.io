List of Obsidian MEDILIG core plugins

  • obsidian-dataview
    • Treat your Obsidian Vault as a database which you can query from. Provides a JavaScript API and pipeline-based query language for filtering, sorting, and extracting data from Markdown pages.
    •  MIT license
    • sponsor this project
  • obsidian-meta-bind-plugin
    • A plugin for Obsidian to make your notes interactive with inline input fields, metadata displays (view fields), and buttons.
    •  GPL-3.0 license
    • sponsor this project
  • templater-obsidian
    • Templater is a template plugin for Obsidian.md. It defines a templating language that lets you insert variables and functions results into your notes. It will also let you execute JavaScript code manipulating those variables and functions.
    •  AGPL-3.0 license
    • sponsor this project

List of plugins that enhance functionality in Obsidian MEDILIG

  • editing-toolbar
    • A plugin that provides a toolbar similar to MS-Word，and adds a minimal and user friendly text editor modal for a smoother writing/editing experience .
    •  MPL-2.0 license
    • sponsor this project
  • file-explorer-note-count
    • This plugin helps you to see the number of elements you have under all available folders within your file explorer. Counts will include total number of all elements under a folder and its sub-folders
  • frontmatter-modified-date
    • This plugin will automatically update a modified date property in your frontmatter/YAML when you edit a note. This is different from other plugins in that it does not use the filesystem modified time. The frontmatter will only update when you actually work on a file inside Obsidian.
    • MIT license
  • obsidian-homepage
    • Open a specified note, canvas, or workspace upon launching Obsidian
    • MIT license
  • local-backup
    • Automatically creates a local backup of the vault.
    • sponsor this project
    • MIT license
  • obsidian-banners
    • An Obsidian plugin to add banner images (and icons) to your notes!
    • MIT license
  • obsidian-custom-attachment-location
    • Customize attachment location with variables($filename, $data, etc) like typora.
  • obsidian-full-calendar
    • This plugin integrates the FullCalendar library into your Obsidian Vault so that you can keep your ever-changing daily schedule and special events and plans alongside your tasks and notes, and link freely between all of them. Each event is stored as a separate note with special frontmatter so you can take notes, form connections and add context to any event on your calendar.
    • MIT license
  • obsidian-hover-editor
    • This plugin enhances the core "Page Preview" plugin by turning the hover popover into a full featured editor instance.
    • MIT license
  • obsidian-image-gallery
    • A zero setup masonry image gallery for Obsidian
    • MIT license
  • obsidian-image-toolkit
    • When you click an image, it will be popped up and you can preview, zoom, move, rotate, flip, invert and copy the image.
    • MIT license
  • obsidian-outliner
    • Work with your lists like in Workflowy or RoamResearch
    • sponsor this project
    • MIT license
  • obsidian-plugin-toc
    • Create a table of contents inside a note
  • obsidian-reminder-plugin
    • This plugin adds feature to manage markdown TODOs with reminder.
    •  MIT license
  • obsidian-style-settings
    • This plugin allows snippet, theme, and plugin CSS files to define a set of configuration options. It then allows users to see all the tweakable settings in one settings pane.
    • it is used to adjust every CSS variable of the Obsidian Minimal theme that is used in MEDILIG
    • GPL-3.0 license
    • sponsor this project
  • obsidian-tasks-plugin
    • Track tasks across your entire vault. Query them and mark them as done wherever you want. Supports due dates, recurring tasks (repetition), done dates, sub-set of checklist items, and filtering.
    • MIT license
    • sponsor this project
  • obsidian-force-view-mode-of-note
    • allows you to indicate through front matter that a note should always be opened in a certain view mode or editing mode.
  • obsidian-zoom
    • Zoom into heading and lists
    • sponsor this project
    • GitHub - vslinko/obsidian-zoom
  • omnisearch
    • It always instantly shows you the most relevant results, thanks to its smart weighting algorithm. Under the hood, it uses the excellent MiniSearch library.
    • sponsor this project
    • GPL-3.0 license
  • recent-files-obsidian
    • Display a list of most recently opened files
    • GPL-3.0 license
    • sponsor this project
  • obsidian-hotkeys-for-specific-files
    • With this plugin, you can open specific files with a hotkey. You can set the files in the settings. A command will be generated for each file, then you can set your hotkey in the default hotkeys section in Obsidian.
    • sponsor this project
  • obsidian-image-captions
    • Add captions to images with inline Markdown and link support. The caption format is compatible with the CommonMark spec and other Markdown applications.
    • GPL-3.0 license
  • obsidian-scroll-to-top-plugin
    • This is a plugin for Obsidian that adds buttons to scroll to the top/bottom of the current note
