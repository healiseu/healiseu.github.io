Add or change the profile photo of the patient

If you click the PHOTO button, Obsidian MEDILIG will search for an image in the predefined media_feeder folder (C:\media_feeder). You must create this folder on your PC as it serves as the source for feeding the patient folder with media files (image, video, text). If this folder is not found, the following message will be displayed.

Click on the message above to dismiss it.

In case Obsidian Vault is installed on another device

It is not possible to transfer files from one device (drive C:\) to another device (drive D:\).

If the folder C:\media_feeder is created but we have not placed any image file in it to update the profile photo, the following message will be displayed.

Click on the message above to dismiss it.

If the C:\media_feeder folder is created, there is an image (avif, bmp, gif, jpeg, jpg, png, svg, webp) placed in it, and your Obsidian Vault is also located on the same drive C:\, the new photo will appear, and the photo field in the front matter will be updated accordingly Fig. 2.
