Windows Share Folder on Local Network

  • Πίνακας ελέγχου (control panel)

    • Προγράμματα > Προγράμματα και δυνατότητες > Ενεργοποίηση ή απενεργοποίηση των δυνατοτήτων των Windows >
      Programs > Programs and Features > Turn Windows features On or Off > SMB 1.0/CIFS
      • Υποστήριξη κοινής χρήσης αρχείων SMB 1.0 / CIFS (checkbox)
    • Δίκτυο και Internet > Κέντρο δικτύου και κοινής χρήσης > Αλλαγή ρυθμίσεων κοινής χρήσης για προχωρημένους
      Network and Internet > Network and sharing center > Advanced sharing settings > change sharing options for different network profiles
      • Ιδιωτικά δίκτυα - Τρέχον προφίλ (Private - current profile )
        • Εντοπισμός δικτύου Ενεργό (checkbox) - Turn on network discovery
        • Αυτόματη ρύθμιση συνδεδεμένων συσκευών δικτύου (chekbox) - Turn on automatic setup of network connected devices
        • Κοινή χρήση αρχείων και εκτυπωτών - Turn on file and printer sharing
    • (Search box) Υπηρεσίες (Services) - Τύπος εκκίνησης Αυτόματα
      • Εντοπισμός SSDP - SSDP Discovery
      • Κεντρικός υπολογιστής υπηρεσίας παροχής εντοπισμού λειτουργιών - Function Discovery Provider Host
      • Δημοσίευση πόρων εντοπισμού λειτουργιών - Function Discovery Resource Publication
      • Κεντρικός υπολογιστής συσκευής τοποθέτησης και άμεσης λειτουργίας γενικής χρήσης (UPnP) - UPnP Device Host

  • Τύπος προφίλ δικτύου - Network Profile

    • από το εικονίδιο WiFi - Διαχείρηση συνδέσεων WiFi
    • Δίκτυο και Internet > WiFi (κάνουμε scroll πάνω δεν φαίνεται στα Windows 11) και εμφανίζονται:
      • ιδιότητες της σύνδεσης
        • Τύπος προφίλ δικτύου - Ιδιωτικό δίκτυο (Network Status - Private Network)
        • Αλλαγή ρυθμίσεων DNS για όλα τα δίκτυα WiFi - Επεξεργασία (button)
          • Επεξεργασία ρυθμίσεων DNS δικτύου > Μή αυτόματα (για σταθερή IP)
          • IPv4 ενεργό - Προτινώμενο DNS 192.168.2.1
          • Προβολή κλειδιού ασφάλειας WiFi - Σύνθετες ιδιότητες δικτύου WiFi
    • Δίκτυο και Internet Network and Internet
      • WiFi (Σύνδεση διαχείριση γνωστών δικτύων)
      • Ιδιότητες υλικού (προβολή διαχείριση ιδιοτήτων) > Ιδιότητες WiFi
        • Αντιστοίχιση διεύθυνσης IP
        • Εκχώριση διακομιστή DNS
        • Περισσότερες επιλογές προσαρμογέα για στατική IP ((change adapter settings for static IP*)
          • WINS με χρήση NetBIOS σε TCP/IPv4 - Enable NetBIOS over TCP/IP

  • Δεξί κλικ πάνω στον φάκελο που θέλουμε να κάνουμε κοινόχρηστο

    • Ιδιότητες (properties)
      • Κοινή χρήση (ΤΑΒ) - Κοινή χρήση για προχωρημένους (Advanced sharing)
        • Κοινή χρήση αυτού του φακέλου (share this folder > share name)
        • Όνομα κοινόχρηστου στοιχείου
        • Δικαιώματα - Δικαιώματα κοινής χρήσης (TAB) - Share permissions (TAB)
          • Ονόματα ομάδων ή χρηστών - Everyone
          • Δικαιώματα (Πλήρης έλεγχος, Αλλαγή) - (Full control, change)
      • Ασφάλεια (TAB) - Security TAB
        • Δικαιώματα > Προσθήκη...
        • Εισάγετε τα ονόματα Everyone (Έλεγχος ονομάτων)
          • Πλήρης έλεγχος
          • Τροποποίηση
net use Z: \\Secretary\myshared
net use Z: \\Secretary\myshared /delete
